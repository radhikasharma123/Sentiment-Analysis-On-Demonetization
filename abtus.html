About The Application

  • This application is mainly developed to create a platform between the government and the citizens of that nation.

  • Opinions are gathered from the users.

  • After gathering the opinions, the analysis team performs "Sentiment Analysis" using the latest technologies.

  • With the help of this application, one can know the impact occurred on the citizens belonging to different occupations and states

  • Government takes decision without considering the opinions of the common man. Therefore, in order to overcome such situations, this application is developed

  • This application is simple to use

  • The user only needs to give the details such as name,age, occupation, state and opinion

  • The rest of the part is handled by the Analysis Team

  • Bigdata technologies are used in order to perform sentiment analysis & Visualization is done using Business Intelligence
