How to use the Application

The user can see the "give your opinion here button" on the right side of the page in the top.
The user needs to give details like name, age, occupation, state and opinion.

In the menu bar which is located in the left side of the page, we can find different options such as gallaery, gathered, About Demonetization, About Us and the help option.

The gallery contains the photos, videos and the news related to demonetization.

In the gathered button, you can find the graphical representation of all the comments that has been collected using this application and also from twitter.

About demonetization option contains all the information related to demonetization.
About us contains the information related to the application and also about the developers of the application

This application is easy to access and can be used in any operating systems.
